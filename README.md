# Amanda Command Center — Prototype

This is a working, mobile-friendly starter prototype.

## What works now
- Dashboard / command-center layout
- Quick capture buttons
- Callahan purchase tracking
- Monthly Callahan totals
- Receipt photo taken / receipt box checklist
- Callahan CSV export
- Business purchase tracking
- Bookkeeping review status
- Business CSV export
- Brain dump
- Tasks
- Manual events
- Manual alerts
- Monthly tax reminder setup
- Morning brief
- Data saved in the browser with localStorage

## Important limitation
This version stores data only in the browser you use. It is a prototype, not yet a secure multi-device app.

## Open it
Double-click `index.html`, or upload all files to a new Replit HTML/CSS/JS project.
